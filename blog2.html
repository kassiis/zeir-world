قالب بصري لبناء العالم — جاهز للتخصيص

قالب أنيق يدعم العربية (RTL) ويحتوي على مكوّنات تُعرض النظام الحاكم، المدارس الفكرية، والموارد السردية. استخدمه كأساس ثم ألصق قالب HTML الخاص بك لأعمل على تكييف التصميم.

نص سريع

ملخص بصري لنظامك: السلطان ← الديوان ← الداي ← الآغا ← بايلم ← زيخ ← مشرف

قم بلصق HTML الخاص بك في المكان المخصص وسأعيد تنسيقه، أو أعطني توجيهات لتخصيص الألوان، الطباعات، والرموز.

مكونات القالب

عرض هرمي

قائمة مرئية لتسلسل السلطة مع أوصاف قابلة للتعديل.

بطاقات المدارس

عرض للمدارس العظيمة: القتالية والفنية والمختلطة، مع حصص ومناهج افتراضية.

موارد قابلة للتحميل

تصدير كـ PDF، JSON للافتراضات، أو assets للرسوم التوضيحية.

نظام الحكم — مخطط هرمي

السلطان
رأس الدولة، حامل لقب "ظلّ الزير".
الديوان
مجلس الشيوخ والاستشارة.
الداي
الوزير الأعظم، يدير الدواوين.
الآغا
القائد العسكري العام.
بايلم
مشرف الإقطاعية.
زيخ
مشرف المحافظة أو شيخ القبيلة.
مشرف
مدير المدينة أو البلدة.

تعليمات سريعة للصق قالبك

  1. ألصق الـ HTML الخاص بك داخل عنصر <main id=\"custom\"> في هذا الملف أو أرسله لي لأقوم بالدمج.
  2. إذا كان القالب يستخدم مكتبات خارجية (Bootstrap, Tailwind) أخبرني لأحافظ على التوافق.
  3. حدد الألوان والخطوط المُفضلة، وسأقوم بتطبيقها على النظام فورًا.
